Interaction notes

• Mobile nav: fixed bottom tab bar (Home/Quests/Party/Notes/More), safe-area padded; top bar collapses to logo + campaign + avatar.

• Tap targets: ≥44px on all interactive elements; HP ± buttons 44×44.

• Editing: every markdown body = rendered view → "✎ Edit" → plain textarea + Save/Cancel. No WYSIWYG in MVP; plain text always valid.

• Role degradation: player = no canon edit buttons, can tick objectives + edit own character; viewer = read-only + quick-capture note only. Enforced by API, mirrored by UI.

• Color semantics are load-bearing: amber=active/DM, emerald=done/positive, rose=danger, violet=AI/lore, sky=threat — consistent across chips, borders, text.

• Keyboard: `/` focuses search (P1), `n` = new note on any page.
